Countries.io is a web application which intended to find the most needed data of a specific legally declared country, conveniently. Currently, this application can fetch information of 250 countries. You can view a live demo in this URLðŸ‘‰ https://countries-io.herokuapp.com/.


<h2 align="center">Data Source</h2>

- The datasource for this project is from https://restcountries.eu/.


<h2 align="center">Technologies</h2>

- This project was generated with `create-react-app`.
- The design is based on Ant Design.
- The maps are from react-leaflet.
- React-redux used for state management.

<h2 align="center">To run</h2>

## Clone the repository

## Use npm to install packages
- run `npm install`.

## Use npm to run the project
- run `npm start`.
